```python
import pandas as pd

df = pd.read_csv('https://raw.githubusercontent.com/Nixtla/transfer-learning-time-series/main/datasets/peyton_manning.csv')
```

## Step 3: Anomaly detection with date features.

You can automatically add date features for anomaly detection using the
`date_features` parameter.

Setting `date_features` to `True` extract all the possible date features
from the dataset.

You can also pass a list of date features you want to create, like
`['days', 'month']`.

```python
anomalies_df_x = nixtla_client.detect_anomalies(
    df, time_col='timestamp', 
    target_col='value', 
    freq='D', 
    date_features=['month', 'year'],
    level=99.99,
)
```
