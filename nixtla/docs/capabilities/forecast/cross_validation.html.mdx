## Step 1: Create an instance of `NixtlaClient`

```python
from nixtla import NixtlaClient
```


```python
nixtla_client = NixtlaClient(
    # defaults to os.environ.get("NIXTLA_API_KEY")
    api_key = 'my_api_key_provided_by_nixtla'
)
```

## Step 2: Load your dataset

```python
import pandas as pd
```


```python
df = pd.read_csv("https://raw.githubusercontent.com/Nixtla/transfer-learning-time-series/main/datasets/air_passengers.csv")
```

## Step 3: Cross-validation

This is done with the `cross_validation` method of `NixtlaClient`.
Simply specify the number of windows for cross-validation using
`n_windows`.

```python
forecast_cv_df = nixtla_client.cross_validation(
    df=df,
    h=12,
    n_windows=2,
    time_col='timestamp',
    target_col="value"
)
```
